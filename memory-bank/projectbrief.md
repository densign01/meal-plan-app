# Project Brief - Meal Plan Assistant

## Project Scope & Goals
Building an AI-powered meal planning application that helps busy families plan healthy, home-cooked meals through conversational interfaces.

## Core Value Proposition
- **Input**: Natural language description of household and weekly schedule
- **Output**: Personalized 7-day meal plan with recipes and organized grocery list
- **Target**: Busy parents who want to cook at home but lack time to plan

## Success Metrics
- Time to first meal plan < 10 minutes
- Weekly active users
- Meal plan completion rate
- Recipe save/favorite rates
- 4-week and 12-week retention

## Constraints
- Dinner-only for MVP (Phase 1)
- No subscription meal kits (self-service planning)
- Family-focused (not single-person optimization)
- Budget-conscious recommendations

## Non-Goals (Phase 1)
- Breakfast/lunch planning
- Grocery delivery integration
- Community recipe sharing
- Nutritional tracking
- Calendar sync
